# HandLog

손글씨를 단어 단위로 녹화하고 동시에 재생해 비디오로 저장하는 브라우저 앱입니다.

## 무엇을 하는 앱인가요?

한 문장을 단어별로 따로 그린 뒤, 모든 단어를 동시에 재생해 문장 전체가 한꺼번에 써지는 것처럼 보이는 영상을 만들 수 있습니다. 녹화는 첫 획이 시작되는 순간 자동으로 시작되고, **완료** 버튼을 누르면 끝납니다. 결과물은 `.webm` 파일로 다운로드됩니다.

## 기능

- **단어별 캡처** — 각 단어를 독립된 트랙으로 녹화
- **동시 재생** — 모든 트랙이 프레임 단위로 동기화되어 함께 재생
- **5가지 펜 종류** — 기본, 압력, 브러시, 캘리그래피, 연필
- **스타일러스 지원** — Pointer Events API를 통해 Apple Pencil 등의 압력·기울임 인식
- **캔버스 설정** — 배경색, 글씨색, 두께 조절
- **배경 이미지** — 이미지를 넣으면 캔버스 크기가 이미지에 맞게 조정
- **레티나 렌더링** — `devicePixelRatio` 스케일링으로 고해상도 디스플레이에서도 선명하게 출력
- **비디오 다운로드** — `MediaRecorder`로 `.webm` (VP9/VP8) 내보내기
- **KO / EN** — 상단 언어 토글

## 사용 방법

1. `index.html`을 브라우저로 열기
2. 배경색, 펜 색상, 두께 설정
3. **단어 추가** 클릭 → 캔버스에 첫 번째 단어 작성 → **완료** 클릭
4. 나머지 단어도 같은 방식으로 반복
5. **재생** 버튼으로 모든 단어가 동시에 그려지는 것을 미리보기
6. **다운로드** 버튼으로 `.webm` 파일 저장

> 팁: 캔버스 녹화는 **첫 획**이 시작되는 순간 자동으로 시작됩니다. 단어마다 타이밍을 자연스럽게 조절할 수 있습니다.

## 펜 종류

| 펜 | 동작 |
|----|------|
| 기본 | 일정한 두께의 부드러운 획 |
| 압력 | 스타일러스 압력에 따라 두께 변화 (마우스 사용 시 고정값 0.5) |
| 브러시 | 부드럽게 테이퍼되는 가장자리, 압력 감도 적용 |
| 캘리그래피 | 납작한 펜촉 — 기울임 각도에 따라 두께 변화 |
| 연필 | 낮은 불투명도로 겹칠수록 진해지는 질감 |

## 기술 노트

**스트로크 스무딩** — 꺾임 현상을 없애기 위해 중점 알고리즘을 사용한 이차 베지어 곡선으로 획을 그립니다. (`moveTo` 중점 → `quadraticCurveTo` 조절점 → 다음 중점)

**재생 방식** — 각 단어는 포인터 이벤트당 `{type, x, y, t, color, width, pressure, tiltX, tiltY, pen}` 데이터를 저장합니다. 내보내기 시 `requestAnimationFrame` 루프가 모든 트랙을 동시에 재생하면서 `MediaRecorder`가 30fps로 캔버스 스트림을 캡처합니다.

**캔버스 크기** — 논리 좌표(`logicalW` / `logicalH`)와 CSS 크기를 분리해 관리합니다. 좌표는 논리 픽셀 기준으로 저장되며, 캔버스 크기가 변경될 때 비례 재조정됩니다. 실제 캔버스 해상도는 `logicalW * devicePixelRatio`로 설정됩니다.

## 요구 사항

`MediaRecorder`와 `canvas.captureStream`을 지원하는 최신 브라우저면 충분합니다. 설치나 빌드 과정 없이 파일을 바로 열면 됩니다.
